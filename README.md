# AirFightMiniProject

Мини-прототип воздушного боя, выполненный как тестовое задание в Unity.  
Игрок управляет самолётом, преследует вражеский самолёт, стреляет по нему, а при потере цели получает подсказку через экранный маркер (референс — сцены преследования из Call of Duty).

## О проекте

Сцена — это небольшой эпизод воздушного боя:

- есть **самолёт игрока**, который летит за вражеским;
- есть **вражеский самолёт**, являющийся основной целью;
- игрок может **стрелять** по врагу, когда тот в зоне видимости;
- если враг выходит за пределы экрана, появляется **экранный маркер**, подсказывающий, где цель и куда разворачиваться.

Фокус проекта — на читаемости цели, работе камеры и удобной навигации по небу, а не на большом количестве контента.

## Основные механики

- **Преследование цели**  
  Самолёт игрока ориентируется на вражеский самолёт и следует за ним, сохраняя динамику полёта.

- **Стрельба по врагу**  
  Игрок может атаковать цель; попадания обрабатываются в игровой логике (урон/состояние врага).

- **Экранный маркер цели**  
  Если вражеский самолёт выходит за пределы поля зрения камеры, на экране появляется маркер, указывающий его направление (в духе современных шутеров).

- **Камера и читаемость пространства**  
  Камера выстроена так, чтобы игроку было удобно ориентироваться в сцене и понимать, где находится цель.
